Random Quote Generator

A simple and elegant Random Quote Generator built using Python Flask, HTML, CSS, and SQLite. The application displays a random inspirational quote each time the page loads or when the user clicks the New Quote button.

Features
Display a random quote on page load
Generate a new random quote with a button click
Show quote text and author name clearly
Modern and responsive user interface
SQLite database for quote storage
Clean and user-friendly design
Lightweight and easy to run locally
Technologies Used
Python 3
Flask
HTML5
CSS3
SQLite

Installation
1. Clone the Repository
git clone https://github.com/your-username/random-quote-generator.git
cd random-quote-generator

Running the Application

Start the Flask server:
python app.py

Open your browser and visit:
http://127.0.0.1:5000
